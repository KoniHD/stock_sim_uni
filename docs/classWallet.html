Wallet::Wallet ( double  funds,
std::unique_ptr< Strategy >  strategy,
std::shared_ptr< StockMarket >  market 
)

Construct a new Wallet object.

The constructor initializes the _funds and _cash_position with funds and _portfolio_value is initially set to 0 but updated when the portflio is assembled using the strategy. The _strategy is moved to be owned by the Wallet. The market is the StockMarket at which the Wallet operates. Since multiple Wallets can operator on this StockMarket, _market is a shared_ptr.

void Wallet::printWalletInfo ( ) const

Prints _cash_position, each owned Stock (value in owned Stock, number of Stocks), and overall _portfolio_value and performance.

The function prints values rounded to two significant digits after dot for _cash_position, and the performance. The other values are printed in their stored precision.
